CRUD-node-js

objetivo:

Fazer um CRUD básico em Node.js me ensinou habilidades fundamentais para o desenvolvimento de aplicativos web. Aprendi a configurar e utilizar um servidor web com Node.js, utilizando a biblioteca Express.js para lidar com rotas e requisições HTTP. A partir disso, pude criar as rotas necessárias para realizar as operações de CRUD (Create, Read, Update e Delete) em um banco de dados. Em seguida, aprendi a utilizar o banco de dados MongoDB, que é uma ferramenta amplamente utilizada em aplicações web modernas. Com o MongoDB, criei um esquema de dados para minha aplicação, definindo as coleções e campos que seriam necessários. Com essas ferramentas em mãos, pude então implementar as funções de CRUD em minha aplicação, utilizando a biblioteca Mongoose.js para interagir com o banco de dados. Isso permitiu que eu criasse, lesse, atualizasse e deletasse dados no meu banco de dados a partir de rotas específicas em minha aplicação.

BOT PARA DISCORD

objetivo:

Criar um bot para Discord me ensinou habilidades fundamentais para o desenvolvimento de aplicativos para esta plataforma. Aprendi a utilizar a biblioteca Discord.js, que é uma das principais bibliotecas utilizadas para criação de bots para o Discord. Com a biblioteca em mãos, pude aprender a interagir com a API do Discord, utilizando a biblioteca para criar uma conexão com o servidor do Discord e implementar as funções que permitiam que meu bot recebesse e enviasse mensagens para os usuários do servidor.
